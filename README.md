# Guided project: foreach / if / array (C#)

English
-------

This repository contains the Starter code and the final solution for the guided project "Develop foreach and if-elseif-else structures to process array data in C#", part of the **Getting started with C#** collection on Microsoft Learn.

Purpose
-------

Practice using `foreach` and `if` / `else if` / `else` constructs to process array data in C#.

Repository structure
--------------------

- `GuidedProject/Starter/` — starter code with instructions and placeholders for the learner to complete.
- `GuidedProject/Final/` — example final solution.
- `*.sln` — Visual Studio / dotnet solution file.

Prerequisites
-------------

- .NET SDK (this repository uses `net8.0` in the project files). Download: https://dotnet.microsoft.com/
- An editor/IDE (Visual Studio, Visual Studio Code, Rider, etc.)

How to run
----------

Open a terminal (PowerShell on Windows) at the repository root and run:

Run the Starter project:

```powershell
dotnet run --project GuidedProject/Starter/Starter.csproj
```

Run the Final project:

```powershell
dotnet run --project GuidedProject/Final/Final.csproj
```

Build the entire solution:

```powershell
dotnet build Guided-project-foreach-if-array-CSharp-main.sln
```

Notes
-----

- The `bin/` and `obj/` directories are created during build and are typically included in `.gitignore`.
- Adjust the `dotnet run` command to target a different framework/version if your SDK differs.
